# AI Image Transformer - Frontend

React + Vite + Tailwind CSS frontend for the AI Image Transformer application.

## Installation

```bash
cd client
npm install
```

## Environment Setup

Create a `.env` file:

```
VITE_API_URL=http://localhost:5000/api
```

## Run Development Server

```bash
npm run dev
```

The app will run on `http://localhost:3000`

## Build for Production

```bash
npm run build
```
